# 舟券アカデミア v86（今日確認分の自動取得アップデート）


## 変更点

- 丸亀・下関の展示データ解析を強化
  - HTML構造が少し変わっても、表の文字列から再解析するフォールバックを追加
  - 展示 / 1周 / まわり足 / 直線 / 体重 / チルトを6艇分チェック
  - 空欄データで既存入力を上書きしないよう修正
- 風向・風速の取得を継続
  - 追い風 / 向かい風 / 左横風 / 右横風に対応
  - 左横風・右横風は 1m / 2m / 3m以上 に自動変換
- 集中アクセス対策
  - 同一の「場・日付・レース」はサーバー側で3分キャッシュ
  - 3分以内にボタンを連打しても、公式サイトへ再アクセスせずキャッシュを返す
  - 自動で3分ごとに取りに行く方式ではなく、必要な時だけ押す方式を維持


- 表示制限対応
  - 画面上の「展示・1周・まわり足・直線」の生数値表示を削除
  - 各艇カードは「コース・チルト・平均ST・F持ち・体重」中心の表示へ変更
  - 展示系データは内部計算のみに使用し、「取得済 / 未取得」の状態表示のみに変更
  - 合算サマリー表は「補正値・合計(直線除く)・差」のみ表示
  - 場平均1着率欄の補足表示を「補正値 / 風補正」中心に変更

- 初期画面を開催場一覧形式に変更
  - 24場をタップできるカード表示に変更
  - タップした時間から次に近いレース番号を自動選択
  - 自動取得対応済みの丸亀・蒲郡・下関はタップ後にそのまま取得開始
  - 未対応場は次R選択だけ行い、展示等の取得は順次対応予定として表示
- 風の扱いを調整
  - 風は非表示にせず表示を維持
  - 自動取得できた場合もプルダウンを残し、急な風向き変化を手動で上書き可能


- 枠別情報ページの一括コピペ対応
  - これまで分かれていた「平均ST」「選手成績」「決まり手/逃げ」を1つのコピペ欄に統合
  - 枠別情報ページを広めにコピーするだけで、枠別成績・平均ST・逃げシミュレーション・決まり手を一括解析
  - 平均STのプルダウンに「直近6ヶ月」「当地」「一般戦」「SG/G1」「F持」を追加
  - 平均STは「今期 / 直近6ヶ月 / 直近3ヶ月 / 直近1ヶ月 / 当地 / 一般戦 / SG/G1 / 初日 / 最終日 / ナイター / F持」に対応

## 注意

## v87 完全自動更新

- 完全自動更新を追加しました。
  - 開催場を選ぶと、締切が近い次Rを自動判定して展示等・風・3連単オッズを取得します。
  - アプリを開いている間だけ、約3分ごとに現在選択中の場で次Rを再判定して再取得します。
  - ブラウザが非表示・バックグラウンドの間は取得を止めます。
  - 画面上で「完全自動更新 ON/OFF」を切り替えられます。
- 公式サイト側への負荷を抑えるため、重複取得を抑制しています。
  - 同じ場・日付・Rの連続取得は30秒以内はスキップします。
  - サーバー側の3分キャッシュは維持しています。
- v86で追加した住之江・蒲郡・下関・丸亀の自動取得対応は維持しています。
